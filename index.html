Penerapan HTML, CSS, dan JavaScript



Sejak diluncurkannya website pertama pada 23 Agustus 1991, website mengalami perubahan yang sangat signifikan. Pada awalnya, website hanya menampilan interface yang apa adanya. Hanya dengan fitur - fitur sederhanaa seperti anchor.

Sekarang, website menjadi platform yang populer, karena menyediakan antarmuka yang menarik, atau biasa disebut dengan UI(User Interface). Banyak sekali pilar - pilar yang dapat membangun website, seperti HTML sebagai kerangkanya, CSS sebagai hiasannya, dan JavaScript sebagai otaknya. Kali ini, kita akan melihat penerapan dari ketiga komponen utama ini. Ayo, tekan tombol di atas untuk melihat effect dari penggunaan JavaScript dalam mengubah attribute dan nilainya.

Udah dicoba blom? Menarik bukan. Masih banyak lagi attribut yang dapat kita ubah secara realtime. Bukan cuman attribut aja, tapi juga bisa mengubah tag, hingga mengubah content. Di bawah masih banyak contoh - contoh penerapan HTML, CSS, dan JavaScript, dan semua itu bisa kita pelajari di Dicoding. Oh iya, mumpung ada beasiswa dari Indosat Ooredoo, yaitu dengan program IDCamp, tunggu apa lagi.... Kuyh daftar di IDCamp. Langsung aja liat contoh lain. Cekidot.

Yuhuuuu. Tulisan itu bisa diubah ubah lo, secara langsung. Mulai dari warna, ukuran, dekorasi teks, align dan juga font familynya. Dengan memanfaatkan kelebihan JavaScript yag bisa membaca isi dari document dengan syntax document, perubahan terhadap dokumen HTML dapat langsung terjadi. Sehingga transisi yang ada di CSS pun berjalan, dan tada.....

Nah, sekarang sudah tau kan. Bahwasanya sebuah website tidak hanya menampilkan kumpulan kata - kata dan gambar, namun juga dapat menampilkan interface yang menarik dan enak dipandang oleh user. Untuk menciptakan tampilan yang User Friendly, harus banyak belajar dan mencari refrensi baik itu dari website lain, dari buku, atau dari tutorial.

JavaScript sendiri, selain bekerja pada client-side, sebenarnya juga dapat bekerja pada server-side. Dimana javaScript ini dapat menggantikan peran PHP dalam menghubungkan data dengan database. Hmm... Masih banyak ya, yang belum kita tahu.

Pada dasarnya, HTML, CSS, dan JavaScript ini adalah sedikit dari komponen utama pembangun website. Unsur - unsur ini hanya berperan dalam Front-End Web, yaitu UI/UX(User Interface/User Experience). Nah, kalau soal database gimana dong? Itu semua terjadi di belakang layar, atau lebih tepatnya pada sisi server side. Disini, kita akan lebih banyak menemukan syntax dari bahasa pemograman yang sebenarnya. Juga lebih membutuhkan keterampilan dalam memecahkan masalah. Nah, sisi yang satu ini disebut Back-End Web.

Untuk bahasa pemogramannya, yang sangat populer adalah PHP. Di mana PHP berbasis OOP atau Object Oriented Programming. Dengan konsep ini, lebih mudah dalam melakukan suatu pekerjaan, membuat sebuah modul untuk dapat dipakai kembali dan lainnya. Contohnya saja, PHP memiliki PDO, yaitu PHP Data Objects. PDO merupakan sebuah Object atau interface universal yang disediakan oleh PHP untuk menjalin hubungan dengan Database.

Terakhir, dalam dunia kerjanya sendiri, banyak sekali pembagian dalam platform Website ini. Namun tiangnya ada tiga, yaitu Front-End Developer sebagai pengembang dalam UI/UX, Back-End Developer sebagai pengembang di sisi server side, dan yang terakhir adalah gabungan dari keduanya, yaitu Full-Stack Developer. And I Wanna Be Full-Stack Developer. Terima Kasih.
